# Images Directory

This directory is for storing restaurant images.

## Recommended Images

- `hero-bg.jpg` - Hero section background image
- `logo.png` - Restaurant logo
- Menu item images
- Team member photos
- Restaurant interior/exterior photos

## Image Guidelines

- Use optimized images for web (compress before uploading)
- Recommended formats: JPG for photos, PNG for logos with transparency
- Maintain aspect ratios appropriate for the design
- Use descriptive filenames
